# BizAxl Projects

Project & Timesheet management for BizAxl ERP.

## Requires
- bizaxl_erp

## DocTypes
- BA Project, BA Task, BA Timesheet
- BA Activity Type, BA Project Type, BA Project Update

## Installation
```bash
bench get-app https://github.com/kevin-kuriakose/bizaxl-projects.git
bench --site yoursite install-app bizaxl_projects
bench --site yoursite migrate
```
